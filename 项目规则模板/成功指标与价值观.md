
## 成功指标与评估标准
- **业务指标**：
  1. 付费客户数量增长率（季度/年度）
  2. 客单价提升幅度（季度/年度）
  3. 客户续约率（尤其是企业版客户）
  4. 免费版到付费版转化率
  5. 付费版内部升级率（标准版到企业版）

- **产品指标**：
  1. 核心功能使用率
  2. 用户满意度（NPS）
  3. 产品迭代速度与质量
  4. 客户支持请求数量与解决时间

- **AI工具评估标准**：
  1. 数据准确性和预测精度
  2. 决策建议的实用性和可执行性
  3. 工具使用的便捷性和集成度
  4. 对业务指标的实际改善效果
  5. 学习能力和持续优化潜力

## 价值观与工作风格
- **价值观**：
  1. 以客户为中心，注重解决实际业务问题
  2. 数据驱动决策，重视客观证据
  3. 结果导向，关注业务目标达成
  4. 持续学习，拥抱新技术和方法
  5. 团队协作，重视跨部门沟通

- **工作风格**：
  1. 结构化思维，善于分解复杂问题
  2. 注重效率，倾向于自动化重复性工作
  3. 开放心态，愿意接受新工具和方法
  4. 务实作风，关注解决方案的落地可行性
  5. 前瞻性思考，主动识别市场机会
